# 🌸 Iris Flower Classification Dashboard using Machine Learning

An interactive Machine Learning web application built using Python and Streamlit to predict the species of an iris flower based on user input features. The application also includes an interactive visualization dashboard for exploring dataset insights.

---

## 📌 Project Description

This project predicts iris flower species using a Logistic Regression machine learning model trained on the Iris dataset.

Users can enter flower measurements such as:

• Sepal Length  
• Sepal Width  
• Petal Length  
• Petal Width  

The system predicts whether the flower belongs to:

✔ Setosa  
✔ Versicolor  
✔ Virginica  

The project also provides interactive visualizations for better understanding of feature distributions and species comparison.

## 📊 Dataset Information

The Iris dataset is a well-known dataset in machine learning used for classification tasks.

It contains measurements of three iris species:

• Setosa  
• Versicolor  
• Virginica  

Each record includes four features:

• Sepal Length  
• Sepal Width  
• Petal Length  
• Petal Width

## ▶️ How to Run the Project

Step 1: Clone the repository

git clone https://github.com/your-username/iris-ml-project.git


Step 2: Open project folder

cd iris-ml-project


Step 3: Install dependencies

Step 4: Run Streamlit application
